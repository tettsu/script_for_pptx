# deleteMetaFromAll.vbs	

指定ディレクトリ内のpptxのメタ情報を全て削除する。

# replacementAll.vbs

指定ディレクトリ内のpptx内の、指定文言を一括置換する。

# 注意

"コメント"は消えるが、"ノート"は消えない
